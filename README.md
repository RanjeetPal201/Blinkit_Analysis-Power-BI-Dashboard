# Blinkit_Analysis-Power-BI-Dashboard (Power BI)
## 📌 Project Overview  
This project is an interactive sales dashboard created using Power BI to analyze Blinkit's retail performance. The dashboard provides meaningful insights into sales trends, item categories, outlet performance, and customer ratings.

It is designed to help businesses make data-driven decisions by visualizing key metrics in a simple and effective way.

---

## 🎯 Objectives  
- Analyze total and average sales performance  
- Understand item category contribution  
- Compare outlet performance based on size, type, and location  
- Evaluate customer ratings and product visibility  
- Identify trends in outlet establishment over time  

---

## 📊 Dashboard Features  

### 🔹 Key Metrics (KPIs)  
- Total Sales: $1.20M  
- Average Sales: $141  
- Number of Items: 8523  
- Average Rating: 3.9  

---

### 🔹 Visual Insights  
- 📈 Outlet Establishment Trend (Year-wise growth)  
- 🥗 Fat Content Analysis (Low Fat vs Regular)  
- 🛍️ Item Type Distribution  
- 📍 Outlet Location Analysis (Tier-wise)  
- 🏪 Outlet Type Breakdown

## 📈 Key Insights

- Regular fat products contribute more revenue  
- Fruits and snacks are top-selling categories  
- Supermarket Type 1 has the highest sales  
- Sales peaked around 2018
